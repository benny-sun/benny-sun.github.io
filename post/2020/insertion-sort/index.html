又叫插入排序，屬於穩定排序。

像是玩撲克牌整理牌組，將未排序的牌堆插入已排序的牌堆那樣，如果在大部分的元素已是有序的話使用此排序法效率較好。

function insertionSort(array $nums) {
    $len = count($nums);
    for ($i = 1; $i < $len; $i++) {
        $temp = $nums[$i];
        for ($j = $i; $j > 0 && $nums[$j - 1] > $temp; $j--) {
            $nums[$j] = $nums[$j - 1];
        }
        $nums[$j] = $temp;
    }
    return $nums;
}
